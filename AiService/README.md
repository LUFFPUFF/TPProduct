Простой API для генерации и коррекции текста на русском языке.  
*Текущая реализация — базовая, в будущем планируется доработка функционала.*

**`На 27.04.25 не собран образ`**

---

## Возможности

1. **Исправление ошибок** в тексте (орфография, пунктуация, грамматика)
2. **Генерация текста** с заданными параметрами (тональность, стиль)
3. Проверка работоспособности сервиса (в разработке)

---

## Использование API

### Исправление ошибок в тексте
**Endpoint:**  
`POST http://localhost:8000/generate`

**Пример запроса:**
```json
{
  "prompt": "Исправь орфографические, пунктуационные и грамматические ошибки в следующем предложении, строго соблюдая нормы современного русского языка. Дай только исправленный вариант без пояснений: Привет анрей я вчира пришол домой",
  "temperature": 0.7,
  "max_new_tokens": 150,
  "top_p": 0.85,
  "do_sample": false,
  "stream": false,
  "is_text_generation": true
}
```

### Генерация текста
**Endpoint:**  
`POST http://localhost:8000/generate`

**Пример запроса:**
```json
{
  "prompt": "Перепиши следующий корпоративный ответ, сделав его более дружелюбным, теплым и естественным, но при этом не перегруженным деталями. Сделай так, чтобы ответ звучал приветливо и было ощущение личного общения. Также убедись, что ответ остается коротким и по делу. Шаблонный ответ: Мы находимся в Москве. Дай только измененный вариант.",
  "temperature": 0.7,
  "max_new_tokens": 150,
  "top_p": 0.85,
  "do_sample": false,
  "stream": false,
  "is_text_generation": true
}
```

### Проверка работоспособности
**Endpoint:**  
`GET http://localhost:8000/health`
